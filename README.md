## DeepMoIC: Multi-Omics Data Integration via Deep Graph Convolutional Networks for Cancer Subtype Classification

### Quick start

```
git clone git@github.com:JiechengWu/DeepMoIC.git
```
Download the latent_data and SNF_fused_matrix.csv from https://figshare.com/s/b931139782497d5da6fd, and put them in then result/TCGA, then
```
python main.py
```



### Requirements

python == 3.7.8

pytorch == 1.10.2

numpy == 1.21.6

pandas==1.3.5

scikit-learn==0.20.0

snfpy==0.2.2


### Reference
